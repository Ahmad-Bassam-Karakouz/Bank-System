## 🚀 Features

* Secure Login System
*  User Permissions
* Full CRUD for Clients
* Deposit & Withdraw operations
* User Management System

🧠 Technologies Used
* C++
* STL (vector, string)
* File Handling (fstream)
* Bitmask Permissions
* Console UI

⚙️ System Flow

1- User logs in

2- System checks permissions

3- Main menu is displayed

4- User performs operations

5- Data is saved in text files

📂 Data Storage
* clients.txt
* users.txt
