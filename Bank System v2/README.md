# 💳 Banking System (OOP Version)

This is the **second version** of the banking system project, redesigned using **object-oriented programming (OOP)** principles to improve its structure, maintainability, and scalability.

## 🚀 Improvements

Compared to version 1, this version includes:

- Better code organization using classes
- Improved readability and maintainability
- Separation of responsibilities
- A more scalable structure for future features

## 🚀 New features in this version:

* Object-oriented design
* Password protection stored in files
* System login logs
* Logging transactions occurring within the system in a separate file
* Conversion to and from all types of currencies

## 🧠 Technologies used

* C++ language
* Object-oriented programming (OOP)
* STL library (vectors, strings)
* File processing (fstream)
* Bitmask permissions
* Command-line user interface
