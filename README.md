# نظام إدارة عملاء البنك Bank System

نظام بسيط لإدارة عملاء البنك، يعمل عبر سطر الأوامر، ومبني باستخدام لغة C++.

يتيح البرنامج إدارة عملاء البنك باستخدام ملف نصي كقاعدة بيانات.

## الميزات

* عرض جميع العملاء
* إضافة عميل جديد
* حذف عميل موجود
* تحديث بيانات العميل
* البحث عن عميل برقم الحساب

## بنية بيانات العميل

يحتوي كل عميل على:

* رقم الحساب
* الرمز السري
* اسم العميل
* رقم الهاتف
* الرصيد

يتم تخزين جميع البيانات في ملف نصي (`clients.txt`) باستخدام فاصل مخصص.


## التقنيات المستخدمة

* لغة C++
* مكتبة STL (`vector`, `string`)
* معالجة الملفات (`fstream`)
* واجهة سطر الأوامر

## قائمة البرنامج

```
[1] عرض جميع العملاء
[2] إضافة عميل جديد
[3] حذف عميل
[4] تحديث بيانات عميل
[5] البحث عن عميل
[6] الخروج
```

## آلية العمل

يقرأ البرنامج بيانات العملاء من الملف `clients.txt`، ويُجري العمليات في الذاكرة باستخدام المتجهات، ثم يكتب التحديثات إلى الملف.

## هدف المشروع

تم إنشاء هذا المشروع للتدرب على:

* معالجة الملفات في لغة C++
* استخدام الهياكل والمتجهات
* تصميم تطبيقات سطر الأوامر
